jeNotes 1.0

API created with Nodejs, Express, MongoDB, and Mongoose.

You can add notes and TODOs.


Routes >
- /api/notes (GET, POST, DELETE)
- /api/notes/:id (GET, DELETE, PUT)
- /api/todos (GET, POST, DELETE)
- /api/todos/:id (GET, DELETE, PUT)

How to run the project?

You need to install Nodejs and MongoDB.

Then > 

In root folder

- run: npm install
- run: npm start
- run: mongod -dbpath data
